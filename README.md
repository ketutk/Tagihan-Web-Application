# Tagihan-Web-Application

This was my first MERN project
If you want to see the live version plis go to this link :
https://krisnacrud.netlify.app

**Language used :**
- HTML 
- CSS 
- JAVASCRIPT

**Framework used :**
- Bootstrap 
- REACTJS 
- NODE JS
- EXPRESS JS

**MODULE USED**
- JSONWEBTOKEN
- BCRYPTJS
- NODEMAILER
- MONGOOSE

- CRUD TAGIHAN
- USER AUTHENTICATION & AUTHORIZATION
- NODEMAILER INTEGRATION
